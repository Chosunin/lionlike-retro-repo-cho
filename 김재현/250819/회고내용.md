# 📚 오늘 학습 회고

---

## 1️⃣ 오늘 배운 것 중 가장 기억에 남는 것 3가지

1. **형 변환은 연관성이 있어야 가능하다**  
   예) `byte → int`

2. **printf를 사용하면 포맷팅이 가능하다**  
   - 주요 서식 지정자:
     - `%s` : 문자열
     - `%d` : 정수
     - `%f` : 실수
     - `%c` : 문자
     - `%b` : 논리값
     - `%n` : 줄바꿈 (OS 독립적)

3. **switch 문에서 break를 호출하지 않으면**  
   해당 case부터 끝까지 모든 case가 실행되므로 주의

---

## 2️⃣ 오늘 배운 것 중 가장 어렵거나 헷갈렸던 것 3가지

1. **삼항연산자**  
   - 조건식이 `true` → `?` 뒤의 값 반환  
   - 조건식이 `false` → `:` 뒤의 값 반환

2. **형 변환 우선순위**  
   `byte → short → int → long → float → double`  
   (작은 범위 → 큰 범위)

3. **switch문에서 사용 불가능한 타입**  
   `long`, `float`, `double`, `boolean`

---

## 3️⃣ 오늘 수업과 회고시간에 대한 느낀점

- 수업 내용은 간단하고, 복습하는 느낌이었다.  
- 회고시간은 나를 위해서라도 적극적으로 참여하는 것이 중요하다는 생각이 들었다.
