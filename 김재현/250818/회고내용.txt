1. 자바 언어의 특징 3가지
 - 객체지향 언어이다. (재사용성, 유지보수성 굿)
 - JVM을 통해 실행되므로 플랫폼의 구애를 받지 않는다.
 - 가비지컬렉션을 통해 사용하지 않는 객체를 자동으로 삭제해준다.
2. JDK 구성요소 3가지
 - JRE (자바 실행환경) / 개발도구 / 문서
3. IDE란?
 - 개발환경 (개발을 편리하게 도와주는 도구)
4. 식별자 규칙
 - 클래스명 : 대문자 시작, 파일명과 일치
 - 단어마다 대문자로 시작하는 낙타표기법 사용
 - 언더바, $ 이외의 기호 사용 불가
5. 핵심요소 3가지
 - 클래스 선언, main메서드, 출력문
6. 주석을 사용하는 이유
 - 클래스나 메서드, 변수등에 붙여서 개발자들이 한 눈에 알아보기 쉬우려고
7. 한줄 주석: // , 여러줄 주석: /* */, JavaDoc 주석: /** */
8. JavaDoc의 주요 태그
 - param : 매개변수 설명 / return : 반환값 설명 / throws : 예외 설명
9. 중요하다고 생각하는 부분: 주석
-> 종류가 여러가지인 것 부터 중요한 것 같다. 쓰는 사람은 귀찮지만 보는 사람은 편하다.
